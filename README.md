# DCoder — Personal Portfolio

> A clean, terminal-inspired frontend portfolio built with pure HTML, CSS, and JavaScript.

---

## 🖥️ Live Demo

[View Portfolio →](https://coodexii.github.io/dalal_portfolio.)

---

## ✨ Features

- **Terminal aesthetic** — monospace fonts, code block, typing animation
- **Dark / Light mode** — saved in localStorage
- **Arabic / English** — full RTL support with Tajawal font
- **Scroll animations** — fade-in and slide effects via IntersectionObserver
- **Glitch effect** — on the hero title hover
- **Fully responsive** — mobile, tablet, and desktop
- **No frameworks** — pure HTML, CSS, and JavaScript only
